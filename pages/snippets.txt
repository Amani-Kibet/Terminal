Method/Filter	Description
-username	Filter by username
-phone	Filter by phone
-course	Filter by course
-year	Filter by year
.sort("a")	Sort ascending by the last filter (e.g., -username)
.sort("z")	Sort descending by the last filter
.limit(n)	Show only n rows
.measure(">n")	Filter rows where the last filter value > n (numeric comparison)
.include("abc")	Show rows where any column in filters contains "abc"
.format()	Make output colorful, alternating row colors, and visually appealing
